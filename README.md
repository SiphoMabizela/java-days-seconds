# Days to Seconds Converter

A simple Java project that converts days into seconds.

## 📌 Description
This project prompts the user to enter a number of days and then calculates the equivalent number of seconds.  
It’s a beginner-friendly Java program demonstrating input handling and basic arithmetic operations.

## ⚙️ How It Works
- The program asks the user for the number of days.
- It multiplies the input by `86400` (the number of seconds in a day).
- It prints the result.
